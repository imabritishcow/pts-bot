# ptsbot
A simple Discord bot to track points for users using MongoDB.

## Example config.json

```
{
  "MONGOURL": "YOUR MONGODB CONNECTION URL",
  "TOKEN": "YOUR DISCORD BOT TOKEN",
  "BOT_ADMINS": ["YOUR DISCORD ID", "OTHER DISCORD ID"]
}
```
Other Discord ID is optional.

Bot admins can use:
* addpts
* delac
* setpts
* staffhelp
